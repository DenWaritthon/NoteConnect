# Phase 6: Graph, Polish & Production Readiness

## Goal

Complete the visual thinking layer and polish frontend quality for production readiness.

## Main Work

- Build `/folders/:folderId/graph`.
- Display graph nodes from notes.
- Display graph edges from relations.
- Support zoom, pan, and drag.
- Support selected node and selected relation details.
- Connect graph interactions to note detail and relation detail pages.
- Build a fullscreen graph modal or layout for mobile.
- Add fallback UI for empty graphs and graph load failures.
- Review performance for large datasets.
- Complete final responsive QA.
- Review accessibility.
- Run lint and build.

## Expected Outcomes

- Users can explore the folder knowledge graph.
- The graph does not recompute relations in the frontend.
- The mobile graph works with touch-friendly controls.
- The frontend passes quality checks before production handoff.

## Out of Scope

- Real-time collaboration.
- A full authentication system unless an approved architecture exists.
- Future features such as tags, summarization, or advanced search.
